How Omni uses AI

Voice is still handled in the browser (Chrome Web Speech). When the option below is on, your spoken command is sent to the HoverArt Node backend you run locally, which calls OpenAI (same OPENAI_SECRET as Two Truths) to turn the phrase into a small list of safe browser actions. The extension then runs those steps; your API key never ships inside the extension package.

Claude / Anthropic is not used here. Turn off the planner to fall back to simple keyword rules only.

Omni planner (OpenAI via backend)

Being logged into HoverArt does not populate the extension by itself. Omni copies hoverart_token into extension storage when it finds any open browser tab matching the HoverArt site URL above (after you log in), or paste here / use HoverArt Omni → Copy JWT.

Start the backend from hover-art-backend with npm start. Test: GET /api/omni/health.
